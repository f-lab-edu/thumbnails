# Thumbnails

Simple game to guess video with more view counts using Youtube thumbnails

Youtube 썸네일을 단서로 진행하는 게임입니다.

- Pick One: Youtube 썸네일로 조회수 높은 영상 맞추는 게임
- Match All: 키워드와 관련된 Youtube 썸네일들을 모두 고르는 게임
